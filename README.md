# Himanshu Lokhande – Portfolio Website

A modern, minimal, and fully responsive personal portfolio showcasing my work, projects, skills, and journey as a Computer Science student and developer.

---

## Features

- Clean and elegant UI with smooth animations  
- Dark and Light theme toggle  
- Side navigation (desktop) + pill-style navbar (mobile)  
- Sections for:
  - Home
  - About
  - Education
  - Projects
  - Skills
  - Achievements
  - Contact
- Responsive design for all screen sizes  
- Typewriter effect and reveal-on-scroll animations  
- Contact form UI included  

---

## Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript**  
- Custom responsive design  
- CSS animations and transitions
